# Otsukai(お使い) — AIにブラウザの用事を頼む

`npm run otsukai -- "頼みたいお使い"` と打つだけで、AIがあなたの代わりに
Webブラウザで用事を片づけます。調べもの・比較・予約の直前まで。

```bash
npm run otsukai -- "食べログで新宿の寿司を評価順に3件、予算と定休日つきで調べて"
npm run otsukai -- "航空券サイトで来週金曜の羽田→福岡の最安を予約直前まで進めて"
npm run otsukai -- "このECサイトで〇〇の在庫と最安値を調べて"
```

---

## 一番大事な約束:「取り消せないこと」はやらない

Otsukai は **優秀だけど権限のないお使い係**として設計されています。

- ✅ **やること**: 見る・調べる・探す・比較する・入力の下書き・予約や購入の**直前まで**進める
- 🛑 **絶対にやらないこと**（あなたの許可なしには実行しない）:
  - 予約の確定 / 申込・応募の送信 / 購入・決済・支払い
  - メッセージ送信 / 投稿・公開 / 予約・注文のキャンセル / 削除 / 退会

これらの「取り消せない操作」は、**実行の直前の画面まで進めて手を止め**、
「ここまで来ました。あとはこのボタンを押せば完了です」とあなたに報告します。
最後のひと押しは、いつも人間がやる。だから安心して任せられます。

## 最後の報告(必ずこの3点)

1. **やったこと** — 訪れたページ、調べて分かったこと（価格・空き・比較結果など具体的に）
2. **いま止まっている地点** — どの画面で、あと何を押せば完了するのか
3. **あなたが判断・実行すべきこと** — 取り消せない操作は必ずここに回される

## セットアップ

Shain 本体と同じです（[README](../README.md) の「必要なもの」を参照）。

- Node.js 18 以上
- 認証: Claude Code のログイン（`npm run otsukai` はこれを使います）または `ANTHROPIC_API_KEY`
- 初回実行時、ブラウザ操作エンジン（`@playwright/mcp`）の取得に少し時間がかかります

## 設定(環境変数)

| 変数 | 既定値 | 説明 |
|---|---|---|
| `OTSUKAI_MODEL` | `sonnet` | 使用モデル。ブラウザ操作は sonnet で十分 |
| `SHAIN_AUTH` | (なし) | `login` で Claude Code ログイン認証を使う（`npm run otsukai` は自動で付与） |

## 仕組み

Shain の基盤(Claude Agent SDK + Playwright MCP)の上で、**お使い1件を最後まで
遂行して報告する単一エージェント**として動きます（`src/otsukai.ts`）。
「お使いの作法」（見る=自由、取り消せない操作=直前で停止）をシステムプロンプトに
焼き込んであるため、依頼文が雑でも安全側に倒れます。

## 注意

- ログインが必要なサイトで認証情報が与えられていない場合、その手前で止まって報告します
  （認証情報を勝手に推測・入力しません）。認証をプロンプトに書くと履歴に残る点に注意。
- 各サイトの利用規約(自動アクセスの可否)は利用者の責任で確認してください。
- 実運用では、まず「調べるだけ」の用事から試すのがおすすめです。
